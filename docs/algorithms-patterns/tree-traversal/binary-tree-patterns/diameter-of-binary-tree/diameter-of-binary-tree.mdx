### Introduction

The **Diameter of Binary Tree** pattern is a common binary tree template used in coding interviews.

---

### Problem Statement

Return the diameter of a binary tree (longest path in number of edges).

### Approach

Compute height of each subtree and update global diameter with `leftHeight + rightHeight`.

```java icon="code" title="Binary Tree - Diameter"
private int diameter = 0;

public int diameterOfBinaryTree(TreeNode root) {
    height(root);
    return diameter;
}

private int height(TreeNode node) {
    if (node == null) return 0;

    int leftHeight = height(node.left);
    int rightHeight = height(node.right);

    diameter = Math.max(diameter, leftHeight + rightHeight);

    return 1 + Math.max(leftHeight, rightHeight);
}
```

### Complexity Analysis

* **Time:** $O(n)$
* **Space:** $O(h)$
