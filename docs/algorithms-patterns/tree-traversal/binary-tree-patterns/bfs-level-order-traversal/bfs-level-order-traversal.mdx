### Introduction

The **BFS Level Order Traversal** pattern is a common binary tree template used in coding interviews.

---

### Problem Statement

Return node values level by level from top to bottom.

### Approach

Use a queue and process nodes in batches using `levelSize = queue.size()`.

```java icon="code" title="Binary Tree - Level Order Traversal"
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int levelSize = queue.size();
        List<Integer> level = new ArrayList<>();

        for (int i = 0; i < levelSize; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);

            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }

        result.add(level);
    }

    return result;
}
```

### Complexity Analysis

* **Time:** $O(n)$
* **Space:** $O(w)$
