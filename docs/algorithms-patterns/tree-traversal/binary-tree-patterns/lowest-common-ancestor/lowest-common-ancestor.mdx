### Introduction

The **Lowest Common Ancestor** pattern is a common binary tree template used in coding interviews.

---

### Problem Statement

Return lowest common ancestor of two nodes `p` and `q` in a binary tree.

### Approach

If current node is `null`, `p`, or `q`, return it. Recurse left and right. If both sides return non-null, current node is LCA.

```java icon="code" title="Binary Tree - Lowest Common Ancestor"
public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
    if (root == null || root == p || root == q) return root;

    TreeNode left = lowestCommonAncestor(root.left, p, q);
    TreeNode right = lowestCommonAncestor(root.right, p, q);

    if (left != null && right != null) return root;

    return left != null ? left : right;
}
```

### Complexity Analysis

* **Time:** $O(n)$
* **Space:** $O(h)$
