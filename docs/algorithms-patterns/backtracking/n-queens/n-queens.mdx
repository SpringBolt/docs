title: Backtracking - N Queens Pattern

### Introduction

The **Constraint Placement Pattern** places items so that global constraints remain valid.

For **N-Queens**, place one queen per row so no two queens share column or diagonal.

---

### Intuition

Process board row by row.

For each row, try every column:

* if column/diagonals are free, place queen,
* recurse to next row,
* remove queen on return.

Using hash sets gives `O(1)` validity checks.

---

### Implementation Example

```java icon="code" title="Backtracking - N Queens"
public List<List<String>> solveNQueens(int n) {
    List<List<String>> result = new ArrayList<>();
    char[][] board = new char[n][n];
    for (char[] row : board) Arrays.fill(row, '.');

    Set<Integer> cols = new HashSet<>();
    Set<Integer> diag1 = new HashSet<>(); // row - col
    Set<Integer> diag2 = new HashSet<>(); // row + col

    dfs(0, n, board, cols, diag1, diag2, result);
    return result;
}

private void dfs(int row, int n, char[][] board, Set<Integer> cols,
                 Set<Integer> diag1, Set<Integer> diag2, List<List<String>> result) {
    if (row == n) {
        result.add(build(board));
        return;
    }

    for (int col = 0; col < n; col++) {
        int d1 = row - col;
        int d2 = row + col;

        if (cols.contains(col) || diag1.contains(d1) || diag2.contains(d2)) continue;

        board[row][col] = 'Q';
        cols.add(col);
        diag1.add(d1);
        diag2.add(d2);

        dfs(row + 1, n, board, cols, diag1, diag2, result);

        board[row][col] = '.';
        cols.remove(col);
        diag1.remove(d1);
        diag2.remove(d2);
    }
}

private List<String> build(char[][] board) {
    List<String> layout = new ArrayList<>();
    for (char[] row : board) {
        layout.add(new String(row));
    }
    return layout;
}
```

---

### When to Use

* Placement/scheduling with mutual exclusion constraints.
* Need all valid board configurations.

### Complexity Analysis

| Metric | Complexity |
| --- | --- |
| Time | Roughly $O(n!)$ (with pruning) |
| Space | $O(n^2)$ board + $O(n)$ recursion |

Pruning via columns and diagonals is essential.
